Gua punya sebuah project website untuk sebuah organisasi sekolah dari SMKN 2 Pinrang yang bernama COM (Community Programmer).

Website ini adalah website dengan style modern dan minimalis.

Website ini responsif untuk segala jenis ukuran device dan juga memiliki animasi interaktif yang lumayan banyak.

Website ini bagusnya dibuat dengan HTML 5 dan Tailwind CSS

Website ini memiliki database yang bernama comsmakda_db

comsmakda_db memiliki tabel users, kas, dokumentasi, projects, agenda, komentar, dan absensi

Website ini punya 2 jenis halaman yaitu public dan role.

Public adalah halaman yang tersedia untuk orang yang belum registrasi sedangkan Role adalah halaman orang yang sudah registrasi dan harus login untuk mengakses halamannya.

Halaman Dashboard ada beberapa role yaitu ketua, sekretaris, bendahara, dan anggota

Halaman Public yang bisa disebut index berisi beberapa section, yaitu :
# Bagian Header
    1. Navbar -> untuk navigation bar
    2. Hero -> untuk tagline atau informasi umum tentang organisasi
# Bagian Content
    3. Tentang COM -> untuk menampilkan mengenai apa itu organisasi COM 
    4. Kenapa Gabung -> untuk menampilkan card yang berisi alasan bergabung ke organisasi COM
    5. Sejarah COM -> untuk menampilkan sejarah terbentuknya organisasi COM dari sekolah SMKN 2 Pinrang
    6. Projects -> untuk menampilkan card yang berisi mengenai project - project website yang pernah dibuat oleh COM
    7. Dokumentasi -> untuk menampilkan card gambar beberapa kegiatan yang telah dilakukan COM
    8. Komentar -> berisi form untuk memberi komentar, kritikan, atau saran untuk organisasi COM
    9. Maps -> berisi tentang lokasi tempat organisasi COM (6J5R+J2G, Jl. Kesehatan, Penrang, Kec. Watang Sawitto, Kabupaten Pinrang, Sulawesi Selatan 91218)
# Bagian Footer
    10. Footer -> untuk menampilkan copyright maupun link sosial media dari COM dan SMKN 2 Pinrang

Ada juga halaman untuk registrasi dan login untuk bisa masuk ke halaman dashboard dengan role - role tertentu

Halaman Dashboard yang bisa disebut index berisi beberapa section, yaitu :
# Bagian Header
    1. Navbar -> untuk navigation bar
# Bagian Content
    1. Banner -> untuk menampilkan sapaan pada user sekaligus rolw nya masing masing
    2. Aktivitas -> berisi card yang dapat ke halaman lain untuk melakukan sesuatu seperti mengelola kas, mengelola absensi, dan lain lain (akan kujelaskan nanti)
